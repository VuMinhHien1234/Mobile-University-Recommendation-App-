University Information & Consultation App
Project Overview

This mobile application is designed to help students explore and learn more about universities, their faculties, and academic programs. It provides an interactive, user-friendly platform for self-assessment and real-time Q&A to support students in making informed decisions about their future education paths.

Main Features
Module 1: Self-Assessment

Displays a comprehensive list of universities across the country.

When users click on a university, they can view detailed information including:

Faculties and available majors

Admission requirements

Campus images and promotional videos

School overview (location, rankings, student life, etc.)

Allows users to compare universities and bookmark their favorite schools.

Module 3: Chat Box

An AI-powered chatbot that provides instant answers to user questions about:

University details (faculties, tuition fees, admission deadlines)

Campus life (clubs, events, dormitories, etc.)

Rules and policies related to education and student life

The chatbot is available 24/7 and supports natural, conversational interaction.
